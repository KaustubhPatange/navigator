Equivalent to shared animations.

You must set a unique android:transitionName on the View of Fragment A to make sure exit shared transition works perfectly.

Parameters

elements

A map of "from View" of Fragment A to the "Transition name of View" of Fragment B.

sharedElementEntering

Set the enter transition for the shared element(s).

sharedElementExiting

Set the exit transition for the shared element(s).

destinationEntering

A transition to be applied on Fragment B. Set -1 to omit.

currentExiting

A transition to be applied on Fragment A. Set -1 to omit.
