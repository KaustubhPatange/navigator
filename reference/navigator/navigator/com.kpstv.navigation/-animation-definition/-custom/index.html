Custom set of animations to play. Supported resource types are "anim", "animator" & "transition".

Parameters

destinationEntering

Animation that'll be played on the destination fragment when navigating from the current fragment to it.

currentExiting

Animation that'll be played on the current fragment when exiting so as to navigate to the destination fragment.

currentReturning

Animation that'll be played on the current fragment when returning from the destination fragment. Default will be the reverse of currentExiting.

destinationExiting

Animation that'll be played on the destination fragment when returning to the previous (current) fragment. Default will be the reverse of destinationEntering.
