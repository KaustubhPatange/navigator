data class PopToFragment(clazz: FragClazz, all: Boolean) : HistoryOptions
Content copied to clipboard

Clear backstack history up to the clazz. This also removes all the necessary fragment involved in that fragment transaction.

  • Example: Consider a history in order [first, second, second, third, forth] where we are navigating to fifth.

  • A PopToFragment(second) call will make history [first, fifth].

  • A PopToFragment(second, all = false) call will make history [first, second, fifth].

Parameters

clazz

Fragment's class that'll be used to identify the entry in backstack.

all

There may be multiple instance of this Fragment, this specifies if we should remove them all?
