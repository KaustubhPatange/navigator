PopToBackStack

data class PopToBackStack(name: String, inclusive: Boolean) : HistoryOptions
Content copied to clipboard

Clear history up to the name of the entry in backstack. This also removes all the necessary fragment involved in that fragment transaction.

  • Example: Consider a history in order [first, second, third, forth]. So a PopToFragment(second) call will make history [first, fifth] where fifth is the fragment to which we are navigating.

The name associated with the entry in backstack.
